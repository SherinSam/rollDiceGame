Game Rules
==========

=> 2 player game
=> Players take turns to click on button to roll the dice
=> Player can do multiple throws in a row until the throw is a zero OR click the HOLD button to add the current score to total score. A zero-throw resets the current score to zero.
=> The player who first reaches a total score of 100 wins!
